01

Drive API 基本概念

  • 簡單直覺：GAS 預設整合 DriveApp 服務，無需管理 API 金鑰。
  • 授權流程：首次執行時系統將自動引導進行雲端硬碟讀寫授權。
  • 迭代器模式：物件搜尋回傳 Iterator，需透過 hasNext() 與 next() 進行遍歷。
var folders = DriveApp.getFolders(); while (folders.hasNext()) { var folder = folders.next(); }
02

檔案上傳、下載和管理

  • 靈活建檔：支援純文字內容、Blob 二進位流等多元建立方式。
  • 快速備份：使用 makeCopy() 迅速建立指定檔案的副本。
  • 安全處置：利用 setTrashed(true) 將檔案先丟入雲端垃圾桶以防誤刪。
var file = DriveApp.createFile('標題.txt', '內容'); file.makeCopy('備份_標題.txt');
03

資料夾結構建立與維護

  • 階層管理：能在指定節點下使用 createFolder() 建立子目錄。
  • 重複防範：建立前先以搜尋語法比對名稱，避免產生無效重複路徑。
  • 多層架構：一鍵自動產出如「企劃、設計、財報、會議」等標準專案目錄。
if (!folder.getFoldersByName("新專案").hasNext()) { folder.createFolder("新專案"); }
04

雲端檔案權限管理

  • 權限細分：區分為編輯者 addEditor() 與檢視者 addViewer()。
  • 外部共享：可動態修改存取級別（如：任何知道連結的人皆可檢視）。
  • 安全回收：利用 removeEditor() 收回不必要的權限。
file.setSharing( DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW );
05

進階檔案搜尋和篩選

  • Query 語法：支援類似 Gmail 搜尋箱的條件式字串篩選。
  • 精確篩選：支援檔名、MimeType 以及是否已移至垃圾桶等條件。
var query = "title contains '發票' and mimeType = 'application/pdf'"; var files = DriveApp.searchFiles(query);
06

自動化備份與同步

  • 排程複製：將指定資料夾內容定時備份至安全的備份區。
  • 防重機制：複製前比對目標區，不重複備份相同檔案。
  • 無痛搬移：使用 moveTo() 搬移檔案至特定資料夾。
file.moveTo(targetFolder);
07

實際應用：收件箱自動歸檔

  • 情境模擬：自動化排程每日深夜掃描不分類收集箱。
  • 年月歸檔：依據「建立時間」動態歸檔至如「2026-05」的子資料夾。
  • 完全自動：設定「時間驅動觸發條件」，完成無人值守自動化。
var year = file.getDateCreated().getFullYear(); var month = file.getDateCreated().getMonth() + 1;
